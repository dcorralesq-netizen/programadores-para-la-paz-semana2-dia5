¿Qué es una ruta (route) en Express?

Respuesta B: Una dirección que define cómo responde el servidor a una solicitud.

¿Qué método HTTP se usa normalmente para consultar información?

Respuesta B: GET.

¿Qué método HTTP se usa normalmente para enviar o registrar información? 

Respuesta A: POST.

¿Qué significa API?

Respuesta B: Interfaz que permite que sistemas o aplicaciones se comuniquen entre sí.
